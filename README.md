# Equity in Healing Scholarship Website

A full-stack scholarship website built with HTML, CSS, JavaScript, and Node.js to support an ongoing scholarship program focused on Black maternal health awareness and student support.

## Features
- Responsive scholarship website
- Node.js backend
- Stripe donation flow
- Custom and preset donation amounts
- Client-side validation and status feedback

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Stripe
